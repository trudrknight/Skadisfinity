# Skadisfinity Layout

Skadisfinity Layout is a web-based calculator for planning 3D printed Skadis-style board layouts.

The app helps builders enter a layout size, select a printer build volume, and see how many printable Skadis boards and trim pieces are needed. It also creates a real-time visual preview so users can understand how the pieces fit together before printing.

## Purpose

Skadisfinity exists to create uniformity amongst Skadis builders. It gives makers a familiar Gridfinity-style calculator experience, adapted for 3D printed Skadis board planning.

## Core Features

- Skadis layout dimension inputs.
- 3D printer build plate selection.
- Default and AU3D print profiles.
- Real-time visual preview.
- Results that list board quantities and physical millimeter dimensions.
- Trim-piece calculation for leftover layout space.
- Support for repeated duplicate layouts.

## Common Use Cases

- Planning a 3D printed Skadis wall.
- Calculating how many Skadis board sections fit on a printer.
- Matching board dimensions to a printer build plate.
- Creating repeatable modular storage layouts.
- Comparing default Skadisfinity sizing with AU3D hole-count sizing.
