# Skadisfinity Layout Attribution

Skadisfinity was inspired by [gridfinitylayout.com](https://gridfinitylayout.com/).

The respective GitHub repository was used to ensure familiarity for users with the exact same Gridfinity design and experience. Skadisfinity adapts that familiar layout-planning experience for 3D printed Skadis builders.

Skadisfinity was born to create uniformity amongst Skadis builders.

## Project Focus

Gridfinity popularized a clear, modular approach to organizing physical spaces. Skadisfinity brings that same spirit of predictable planning and repeatable parts to Skadis-style pegboard layouts.
